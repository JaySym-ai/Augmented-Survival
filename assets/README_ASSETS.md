# Game Assets

This directory contains all game assets organized by type.

## Directory Structure

- `characters/` — Character models and animations
- `buildings/` — Building models (houses, workshops, walls, etc.)
- `props/` — Environmental props (trees, rocks, crates, etc.)
- `terrain/` — Terrain textures and heightmaps
- `textures/` — Shared textures and materials
- `hdr/` — HDR environment maps for lighting
- `licenses/` — License files for third-party assets

## Asset Guidelines

- Use glTF/GLB format for 3D models
- Textures: PNG or KTX2 (compressed), max 2048x2048
- HDR maps: .hdr or .exr format
- All third-party assets must have a license file in `licenses/`
